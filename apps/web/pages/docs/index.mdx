# AI Chat Platform Documentation

Welcome to the comprehensive documentation for the AI Chat Platform. This documentation covers everything you need to understand, develop, deploy, and contribute to the platform.

## What is AI Chat Platform?

The AI Chat Platform is a modern web application that provides an interactive chat interface powered by AI models through OpenRouter API. It features real-time messaging, user authentication, conversation management, and advanced AI model selection.

## Quick Start

If you're new to the platform, start with our [Getting Started guide](getting-started) to set up your development environment and run the application locally.

## Features

- **Real-time Chat**: Instant messaging with AI models
- **Multi-Model Support**: Integration with various AI providers via OpenRouter
- **User Authentication**: Secure login and user management
- **Conversation History**: Persistent chat sessions
- **Advanced Settings**: Model selection and parameter tuning
- **Responsive Design**: Mobile-friendly interface
- **Accessibility**: WCAG 2.1 AA compliant
- **Performance Monitoring**: Built-in analytics and metrics

## Technology Stack

- **Frontend**: Next.js 15, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **AI Integration**: OpenRouter API
- **Deployment**: Vercel
- **Testing**: Jest, Playwright, Lighthouse
- **Documentation**: Nextra (this site!)
